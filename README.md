# Flexboxgrid.scss

## Minimal

Customizable version of [flexboxgrid](http://flexboxgrid.com)

## Usage

set your container and breakpoints variables in `_variables.scss`

```scss
$screen-sm:48em;
$container-sm:46rem;
$screen-md:62em;
$container-md:61rem;
$screen-lg:75em;
$container-lg:71rem;



```
